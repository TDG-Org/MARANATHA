# MARANATHA

A calm, hand-crafted 2D browser game. Play in your browser — no install, no login.

**▶ Play:** https://tdg-org.github.io/MARANATHA/

## Run locally

```bash
npm install
npm run dev      # http://localhost:5173
npm run build    # production build → dist/
```

## Tech

Phaser 3 · Vite · plain ES-module JavaScript. All visuals are generated in code (no image assets), so it stays light and runs on low-end phones and laptops. Scripture text is the Berean Standard Bible (public domain).
